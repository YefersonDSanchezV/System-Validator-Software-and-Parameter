# Imagenes del reporte institucional

Coloca aqui las imagenes oficiales usadas por el PDF del reporte de firmas.

Nombres requeridos:

- logo-acta.png -> logo de cabecera del acta.
- firmas-final.png -> franja institucional final (ELABORO / REVISION TECNICA Y ESTRUCTURAL / APROBO).

Rutas publicas que consume el frontend:

- /report/logo-acta.png
- /report/firmas-final.png

Recomendacion:

- Usa PNG en alta resolucion y con fondo blanco.
- Mantener una proporcion horizontal para firmas-final.png para evitar recortes.
